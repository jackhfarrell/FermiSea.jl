# Reference

This section collects the public API for FermiSea.jl.

The main model is [`IsotropicFermiHarmonics2D`](@ref), which provides the Trixi
equations interface for two-dimensional isotropic Fermi-surface harmonic
transport. Source terms, boundary conditions, diagnostics, and output helpers are
documented alongside it.
